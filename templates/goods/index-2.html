Improving transportation services

This project aims to develop a system that utilizes dynamic and genetic algorithms to improve logistics services in a specific region. The system will be designed to solve the problem of distributing a fleet of vehicles, planning routes, and optimizing transportation costs and time. Each vehicle will have a capacity, and the goal is to distribute goods with a maximum value for each vehicle while traversing through a set of addresses.
